# Practice Problems

Solve these problems to master data structures!

## Difficulty Levels

- 游릭 **Easy**: 5-15 min
- 游리 **Medium**: 15-45 min  
- 游댮 **Hard**: 45+ min

## Arrays

### 游릭 Easy

1. **Reverse an Array**
   - Input: [1, 2, 3, 4, 5]
   - Output: [5, 4, 3, 2, 1]

2. **Find Maximum Element**
   - Input: [3, 7, 2, 9, 1]
   - Output: 9

3. **Check if Array Contains Element**
   - Input: arr = [1, 2, 3], target = 2
   - Output: true

### 游리 Medium

4. **Two Sum**
   - Find two numbers that add up to a target
   - Input: [2, 7, 11, 15], target = 9
   - Output: [0, 1] (indices of 2 and 7)

5. **Remove Duplicates**
   - Remove duplicate elements from array
   - Input: [1, 1, 2, 2, 3]
   - Output: [1, 2, 3]

6. **Rotate Array**
   - Rotate array to the right by k positions
   - Input: [1, 2, 3, 4, 5], k = 2
   - Output: [4, 5, 1, 2, 3]

### 游댮 Hard

7. **Longest Consecutive Sequence**
   - Find longest consecutive elements sequence
   - Input: [100, 4, 200, 1, 3, 2]
   - Output: 4 (sequence: [1, 2, 3, 4])

## How to Practice

1. **Understand** the problem completely
2. **Plan** your solution on paper
3. **Code** the solution
4. **Test** with provided examples
5. **Optimize** if possible
6. **Explain** your approach

## Tips

- Read the problem carefully
- Consider edge cases (empty, single element, duplicates)
- Start with brute force, then optimize
- Test your code with multiple inputs
- Look for patterns and similar problems
